# **App Name**: EZBUY Landing Page

## Core Features:

- Hero Section with Featured Offer: Display a prominent banner with a featured gaming hardware offer (e.g., RTX 4090) and a call-to-action button.
- Featured Products Grid: Showcase a grid of featured gaming products (e.g., Mechanical Keyboards, Graphics Cards, 144Hz Monitors) with details like image, name, discounted price, 'Offer' tag, and 'Add to Cart' button.
- Category Filters: Enable quick filtering by categories like Peripherals, Components, Monitors, and Gaming Chairs using icons or buttons.
- Flash Sales Section: Implement a carousel or row of products with a countdown timer (visual simulation) for limited-time offers.

## Style Guidelines:

- Primary color: Neon Cyan (#00E0FF) for accents and interactive elements to create a 'gamer' aesthetic.
- Background: Deep Black (#050505) for a dark mode base.
- Accent: Magenta (#FF0055) for secondary highlights and gradient combinations with Neon Cyan.
- Body and headline font: 'Inter' sans-serif for a clean and modern look. This is a versatile font suitable for both headlines and body text.
- Use thin, outlined icons in white or light gray.
- Organize the layout in cards with a clean, 'gamer' aesthetic, and maintain consistency across sections.
- Implement hover effects on product cards (illumination or lifting).
